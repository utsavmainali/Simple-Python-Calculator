# Python Calculator

A **modern and responsive Python calculator** built using **Tkinter**, capable of performing basic arithmetic operations with a clean and user-friendly GUI.  
This project demonstrates Python GUI programming and practical application of Tkinter.

---

## Features

| Feature                    | Description                                           |
|-----------------------------|-------------------------------------------------------|
| Addition, Subtraction, Multiplication, Division | Perform basic arithmetic operations |
| Decimal Support             | Handle decimal numbers accurately                     |
| Clear Button (`C`)          | Reset the input easily                                |
| Equals Button (`=`)         | Calculate the result and spans two columns           |
| Responsive Layout           | Buttons and entry resize automatically               |
| Modern Colors & Fonts       | Clean and user-friendly interface                     |
| Resizable Window            | Maximize or resize window; layout adjusts dynamically|
